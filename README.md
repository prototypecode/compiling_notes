# Compiling Notes

---

```bash
// Update Arch

$ sudo pacman -Syu
```

```bash
// Search Database

$ sudo pacman -Ss gcc
```

```bash
// Install GCC

$ sudo pacman -S gcc
```

```bash
// Verify Version

$ gcc --version
```

```bash
// Remove GCC

$ sudo -Rns gcc

// Remove GCC & Dependencies

$ sudo pacman -Rcnu gcc
```
